# **App Name**: GlowEarn

## Core Features:

- User Account Creation: Allows new users to create an account through a secure, multi-field registration form including Name, Mobile Number, Email, Password, and Confirm Password.
- Secure User Authentication: Enables existing users to securely log into their accounts.
- Personalized User Profile: Displays user information with a customizable golden circular avatar and basic profile details, accessible via the bottom navigation.
- Digital Wallet Management: Provides an interface for users to view and manage their earnings and funds, accessible from the bottom navigation.
- Interactive Leaderboard Display: Showcases top-ranking users, encouraging competitive engagement, available through the bottom navigation.
- AI-Powered Earning Recommendations: A tool that analyzes user activity and profile data to provide personalized suggestions for optimizing earning potential within the app.

## Style Guidelines:

- Main Background: A Deep Navy Blue (#081926) sets a sophisticated and immersive dark theme for the application.
- Primary Interactive Elements and Highlights: A vibrant Golden Glow (#FADB3B) is used for input field borders, text, and main button states, providing high contrast and a premium feel against the dark background.
- Accent Header Bar: A rich Forest Green (#407040) gradient is used for the top header bar, offering a subtle yet distinct visual break.
- Headlines and Body Text: 'Inter' (sans-serif) is chosen for its clean, bold, and modern appearance, with text displayed in a mix of golden and soft white for readability on the dark background.
- Icons: A combination of soft white and gold icons is used consistently as leading icons within input fields and for the bottom navigation options. User avatars are displayed as a prominent golden circular element.
- Decorative Elements: Floating golden coins and gears are incorporated into the background, subtly reinforcing the app's 'Earning' and 'Working' nature.
- Input Fields: Designed as rounded rectangular boxes featuring a glowing golden boundary, providing a visually appealing and interactive input experience.
- Navigation Structure: The app features a distinctive Forest Green gradient header bar at the top and a semi-transparent, dark glassmorphism bottom navigation bar containing five key options.
- Buttons: The 'Sign Up Now' button features a high-gloss gold gradient with a subtle shimmering effect, making it highly engaging and premium-looking.
- Background Elements: The floating golden coins and gears include gentle animations, adding a dynamic and lively quality to the application's visual environment.
